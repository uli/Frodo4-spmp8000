What's new in V4.2?

The most important changes from V4.1 are:
  • Changed license to GPL
  • Replaced the Tcl/Tk GUI with one based on GTK+
  • Use SDL for display, sound, and joystick input under Unix
  • Use OpenGL for resizable window display under Unix
  • C64 ROMs are built into the Frodo executable
For the tons of other small changes and bug fixes, please look here.
